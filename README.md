# Memermon
the incredible, ***MEME***tastic discord.js bot

### Meme Format
Memes are composed of a single JSON object, like so:
```json
{
  "name": "Epic meme",
  "image": "epicurl.png",
  "health": 100,
  "protecc": 50,
  "attacc": 50,
  "devolves_into": "Another epic meme"
}
```
If you wish to add a meme, create an issue with a JSON like the one above!
